# 外部库

# threading

用于实现多线程编程的模块

# uvicorn

专为 Python 异步框架（如 FastAPI、Starlette）打造的 ASGI 服务器

# Pillow

是 Python 生态中图像处理的基石，但需要注意的是，我们通常使用的其实是它的一个现代、活跃的分支——Pillow

# pystray
pystray 是一个专门用来在系统托盘（通知区域）创建图标的 Python 库。它就像给你的程序在电脑右下角安个家，让你能通过右键菜单来控制程序，非常适合开发那些需要常驻后台的工具。

核心就是用 Icon 对象把图标和菜单绑定起来，然后调用 run() 方法让它显示在系统托盘里

# PyInstaller

将 Python 程序打包成独立可执行文件的工具。它会把 Python 解释器、你的代码以及所有依赖库打包到一起，生成一个 .exe（Windows）或可执行文件（Linux/macOS）。用户无需安装 Python 就能直接运行
